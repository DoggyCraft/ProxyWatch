## Permissions
```YAML
ProxyWatch.*:
    description: Gives access to all ProxyWatch commands
    default: false
ProxyWatch.reload:
    description: Player can reload the configuration(s)
    default: false
ProxyWatch.save:
    description: Player can save the configuration(s)
    default: false
```
